# プロジェクト概要

## プロジェクトの目的
このプロジェクトは、小学生向けの掛け算学習アプリケーションです。
九九（1の段～9の段）を4択クイズ形式で学習でき、音声読み上げ機能とゲーム履歴機能を備えています。

## 技術スタック
- **フロントエンド**: 純粋なHTML5 + CSS3 + JavaScript (ES6+)
- **音声合成**: Web Speech API (SpeechSynthesisUtterance)
- **ストレージ**: localStorage (設定とゲーム履歴の保存)
- **ビルドツール**: なし（静的ファイルのみ）

## 機能
1. **問題生成**: 選択された段から問題をランダムに出題（重複回避機能付き）
2. **音声読み上げ**: 問題を日本語で読み上げ（Google 日本語音声使用）
3. **4択クイズ**: 正解と3つの不正解選択肢を表示
4. **正誤判定**: 正解時は緑色表示と正解音、不正解時はピンク色表示と不正解音+答えの読み上げ
5. **設定機能**:
   - 問題の範囲選択（1～9の段を個別に選択可能）
   - 問題数設定（1～81問）
6. **履歴機能**: 過去のゲーム結果を日付順に表示（正解数、問題数、メダル表示）

## デプロイ環境
- 静的ファイルとして任意のWebサーバーまたはローカルでブラウザから直接開いて使用
- 外部依存なし（インターネット接続不要、ただし音声合成機能はブラウザに依存）
